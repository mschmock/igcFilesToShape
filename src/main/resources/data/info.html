IGC to Raster Converter

Information

The application extracts the track log from one or multible igc-files.
All file are combined into one single raster with a specific cell size. For each cell the reccurence is calculated. The raster data can be exported as a image (png file) or a raster-file (xyz file format).

  • Input data as IGC-Files (WGS84)
  • Output is working with Swiss reference systems LV95
  • Output limited to Switzerland
